Real-time Privacy Policy

Effective date: April 26, 2026
Extension name: Real-time
Chrome Web Store item ID: fmcjklhddhonhjjnlnnkaogepihkbhhh

Real-time is a Chrome extension that shows the real remaining time of a video based on the current playback speed. The extension does not collect, transmit, sell, or share personal information.

Information We Collect

Real-time does not collect personal information such as names, email addresses, account information, payment information, authentication information, or user-generated content.

To provide its core feature, Real-time reads the current page's video element in the user's browser. It may read the video's current playback time, duration, and playback speed only to calculate and display the real remaining time.

This video playback information is processed locally in the browser. It is not stored, transmitted to a server, or shared with the developer or any third party.

Chrome Storage

Real-time uses the Chrome storage permission only to save the user's on/off setting from the extension popup. This setting is stored through chrome.storage.sync so the extension can remember whether the user enabled or disabled Real-time.

The saved setting does not include personal information, browsing history, video titles, watched content, URLs, or account data.

How We Use Information

Real-time uses locally available video playback information only to provide its single purpose: showing the real remaining video time based on the current playback speed.

Real-time does not use user data for advertising, analytics, tracking, profiling, or any unrelated purpose.

Information Sharing

Real-time does not sell, transfer, disclose, or share user data with third parties. The extension does not send user data to external servers.

Advertising and Analytics

Real-time does not include advertisements, affiliate links, tracking pixels, analytics services, or behavioral measurement tools.

Limited Use Disclosure

Real-time's use of information is limited to providing or improving its single user-facing purpose. The use of information received from Chrome extension APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Data Security

Real-time does not transmit personal or sensitive user data. The extension's processing happens locally in the user's browser.

Changes to This Policy

This policy may be updated when Real-time's features, permissions, or data practices change. Updates will be published on this page.

Contact

For questions about this privacy policy, please use the Real-time GitHub repository: https://github.com/yunkooo/real-time
